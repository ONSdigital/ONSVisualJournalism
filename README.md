# ONS Visual Journalism - quarto
Website for showing the work of publishing in ONS

This repository contains the source materials used for ONS Visual Journalism.

## Software

The materials are built with [Quarto](https://quarto.org/).
Carousels are built with [tomicapretto's quarto-carousel](https://github.com/tomicapretto/quarto-carousel).

## Structure

* Each folder contains a different page. 
* Images for a specific page should go in the `images/{page}` folder, and images to be used in all pages (e.g. logos) should be in `images/`.
